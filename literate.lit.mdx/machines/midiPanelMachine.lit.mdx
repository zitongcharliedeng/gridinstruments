# MIDI Panel Machine

A minimal two-state machine that controls whether the MIDI device panel is visible. A single `TOGGLE_MIDI` event flips between `closed` and `open`.

## Types

The event union has a single member: `TOGGLE_MIDI`. No context is required — visibility is captured entirely by the active state.

``` {.typescript file=machines/midiPanelMachine.ts}
import { setup } from 'xstate';

interface MidiPanelEvent { type: 'TOGGLE_MIDI' }
```

## Machine

Both states respond to `TOGGLE_MIDI` by crossing to the other state. The machine starts `closed`.

``` {.typescript file=machines/midiPanelMachine.ts}
export const midiPanelMachine = setup({
  types: {
    events: {} as MidiPanelEvent,
  },
}).createMachine({
  id: 'midiPanel',
  initial: 'closed',
  states: {
    closed: { on: { TOGGLE_MIDI: 'open' } },
    open: { on: { TOGGLE_MIDI: 'closed' } },
  },
});
```
