# Package Declaration

This file declares the npm dependencies consumed by `bun2nix` inside the nix flake. It is a bootstrap artifact — it must exist at the project root for bun to generate `bun.lock` and `bun.nix`, same as `flake.nix`.

The `scripts` section is minimal — `literate-state-machine-wiki build` handles the full pipeline. These scripts exist only as convenience aliases for developer muscle memory.

## Dependencies

**Runtime:** SolidJS (UI), XState (state machines), Effect-TS (dependency injection), tonal (music theory), roughjs (sketch rendering), overlayscrollbars, lucide (icons).

**Dev:** TypeScript, Vite + SolidJS plugin, ESLint + typescript-eslint + solid plugin, Playwright (testing), ast-grep (structural linting).
